CF1436B 题解

刚开始做这道题的时候, 我是没有任何思路, 直到出题人发送了这样一条提示:

I just want to remind you that:

Smallest non-negative integers are 0,1,2,3,⋯0, 1, 2, 3, \cdots0,1,2,3,⋯

Smallest prime numbers are 2,3,5,7,⋯2, 3, 5, 7, \cdots2,3,5,7,⋯

我这才意识到, 组成质方的数是 非负数 而并非整数, 即可以用 000. 那这题就变简单了, 我们可以只用 000 和 111 来填充这个矩阵.

先考虑一下小点的 nnn, 比如 n=10n = 10n=10. 由于 222 是一个质数, 我们可以把 (1,1),(1,2),(2,1),(2,2)(1, 1), (1, 2), (2, 1), (2, 2)(1,1),(1,2),(2,1),(2,2) 填充为 111, 第 1,21, 21,2 行和第 1,21, 21,2 列的其他格子填充 000, 这样这两行两列的和就都是 222, 符合条件了.

接下来, 由于 1,21,21,2 两行两列已经被填充完成了, 我们不能再去动这两行, 如果我们希望把 3,43,43,4 两行也变成 222, 最简单的方法就是把 (3,3),(3,4),(4,3),(4,4)(3, 3), (3, 4), (4, 3), (4, 4)(3,3),(3,4),(4,3),(4,4) 填充为 111, 这样就可以满足 3,43,43,4 两行两列的条件了.

接下来如法炮制, 填充 5,65,65,6 行, 7,87,87,8 行和 9,109,109,10 行, 矩阵就完成了.

上面的方法只针对与 n mod 2=0n \bmod 2 = 0nmod2=0, 即 nnn 为偶数的情况, 那如果 nnn 等于奇数, 就会出现下图的情况, 最后一行凑不成质数, 而其他格子都已被填充, 这该怎么办呢?

这也很简单, 因为 333 也是一个质数, 我们只需把在 1∼n−31 \sim n - 31∼n−3 行列的格子按照上面的方法填充, 最后把坐标为 (n−3,n−3)∼(n,n)(n - 3, n - 3) \sim (n, n)(n−3,n−3)∼(n,n) 填充为 111 即可.
