CF690A2 题解

考虑有 000 个金币时哪些人会存活。111 号、222 号显然存活，333 号会被 1,21,21,2 反对所以无法存活，444 号会获得 333 的赞同票所以可以存活。可以发现，存活的人是 1,2,4,8,⋯1,2,4,8,\cdots1,2,4,8,⋯。

有 111 个金币时，1,2,3,41,2,3,41,2,3,4 显然存活，555 无法获得多于 222 个选票所以无法存活，666 会获得 555 的赞同票所以可以存活。可以发现，存活的人是 1,2,3,4,6,10⋯1,2,3,4,6,10\cdots1,2,3,4,6,10⋯。

所以得出结论，有 BBB 个金币时，存活的人的集合是 {1,2,3,...,2B}∪(2B+{1,2,4,8...})\{1,2,3,...,2B\}\cup(2B+\{1,2,4,8...\}){1,2,3,...,2B}∪(2B+{1,2,4,8...})。

  • 当 n mod 2=1n \bmod 2 = 1nmod2=1 时，B=(n−1)/2B=(n-1)/2B=(n−1)/2 时 nnn会第一次在集合中出现，答案为 n−12\dfrac{n-1}22n−1​。
  • 当 n mod 2=0n\bmod 2 = 0nmod2=0 时，找到最大的 222 的次幂 ppp 使得 p≤np\le np≤n，此时 nnn 会在 B=n−p2B=\dfrac{n-p}2B=2n−p​ 时第一次在集合中出现，答案为 n−p2\dfrac{n-p}22n−p​。
