CF97C 题解

今天模拟赛正好出了这道题，分享一种考场上写的做法。

考虑每种操作所能增加或减少多少个还能参加一次比赛的人数，我们设第 iii 种操作所增加的人数为 bib_ibi​，则：

bi={ni=1bi−1−2otherwise.b_i=\begin{cases}n&i=1\\b_{i-1}-2 &\text{otherwise}.\end{cases}bi​={nbi−1​−2​i=1otherwise.​

分析样例可得，最优策略一定是一个 bi>0b_i>0bi​>0 的 iii 操作和一个 bj<0b_j<0bj​<0 的 jjj 操作不断交替，然后一直循环下去。根据贪心思想，因为数据保证 pi<pjp_i<p_jpi​<pj​，我们要让 jjj 操作尽量的多。我们要让每次循环后的次数为 111 的人数均不变，换句话说，iii 操作是为 jjj 操作「提供」次数为 111 的人数。计算方程

xbi+ybj=0,xb_i+yb_j=0,xbi​+ybj​=0,

可得出其中一个解是

{x=−bjy=bi.\begin{cases}x=-b_j\\y=b_i.\end{cases}{x=−bj​y=bi​.​

如果按着这个循环不断重复无穷遍，则平均概率为：

−bjpi+bipj−bj+bi.\frac{-b_jp_i+b_ip_j}{-b_j+b_i}.−bj​+bi​−bj​pi​+bi​pj​​.

则答案为：

max⁡i∈{i∣bi>0}max⁡j∈{j∣bj<0}−bjpi+bipj−bj+bi\max_{i\in\{i|b_i>0\}}\max_{j\in\{j|b_j<0\}}\frac{-b_jp_i+b_ip_j}{-b_j+b_i}i∈{i∣bi​>0}max​j∈{j∣bj​<0}max​−bj​+bi​−bj​pi​+bi​pj​​

另外有一种情况需要特判，就是存在一个 kkk 使得 bk=0b_k=0bk​=0 的情况，当 bk=0b_k=0bk​=0 时我们可以只选择一种操作不断重复，平均概率为 pkp_kpk​。这种情况下的答案为 max⁡{原答案,pk}\max\{\text{原答案},p_k\}max{原答案,pk​}。

signed main() {
  n = in;
  int nowone = n;
  rep(i, 0, n) {
    lf p;
    cin >> p;
    if (nowone > 0)
      zheng.insert({nowone, p});
    else if (nowone == 0)
      ans = p;
    else
      fu.insert({nowone, p});
    nowone -= 2;
  }
  each(z, zheng) {
    each(f, fu) {
      lf res = (f.gx * z.one + z.gx * (-f.one)) / (z.one - f.one);
      up(ans, res);
    }
  }
  printf("%.12LF\n", ans);
  return 0;
}
