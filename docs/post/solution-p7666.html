P7666 题解

本题解修改于 @uliahradri 大佬的（未通过）题解。

题目大意

有 nnn 个点，用 kkk 个区间覆盖这些点且使区间长度和最小。

解题思路

随便手玩一下样例，画个图，可以发现这些区间之间存在 k−1k-1k−1 个空。

而有 nnn 个点，所以共有 n−1n-1n−1 个空，点 iii 和点 i+1i+1i+1 之间的空的长度为 ti+1−ti−1t_{i+1}-t_{i}-1ti+1​−ti​−1。

贪心地想，我们找到前 k−1k-1k−1 大的空，那么答案就是 tn−t1+1t_n-t_1+1tn​−t1​+1 再减去这 k−1k-1k−1 个空的长度，找前 k−1k-1k−1 大的空我们可以从大到小排序后贪心。

需要注意的是：某个测试点 n=1n=1n=1，如果写的丑可能需要特判一下。

参考代码

signed main() {
  in(n)(k)(t, n);
  re(i, n - 1) u[i] = t[i + 1] - t[i] - 1;
  sort(u + 1, u + n, [](int a, int b) { return a > b; });
  ans = t[n] - t[1] + 1;
  re(i, k - 1) ans -= u[i];
  out(ans)('\n');
  return 0;
}
